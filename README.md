# 🕌 Prayer Times App

A simple web application that displays daily prayer times based on city using a public API.

---

## 🚀 Features

- 📍 Get prayer times by city
- 🕌 Displays all prayers (Fajr, Dhuhr, Asr, Maghrib, Isha)
- ⏰ Updates automatically based on current date
- 🌍 Supports different cities


---

## 🛠️ Built With

- HTML / CSS / JavaScript 
- Axios
- Aladhan API
